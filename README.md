# App Educacional

Este é um aplicativo educacional completo com backend Node.js, frontend web Next.js e mobile React Native com Expo.

## Estrutura

- backend: API REST com Express e MongoDB
- web: Aplicação web com Next.js
- mobile: Aplicação mobile com React Native e Expo
- shared: Código compartilhado (modelos e utilitários)

## Como executar

### Backend

cd backend
npm install
npm run dev

### Web

cd web
npm install
npm run dev

### Mobile

cd mobile
npm install
expo start

Certifique-se de ter MongoDB rodando localmente.
